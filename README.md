# Prisoner's dilemma simulation

An object-oriented simulation of the Iterated Prisoner's Dilemma with multiple strategies, tournament mode, and strategy evaluation against a Random player.

## Features

- 10-round match simulation
- Full round-robin tournament between selected players
- Average score calculation per strategy
- Strategy vs Random player duel mode
- Modular architecture (players / tournament / config / main)

## Strategies

- Cooperator
- Defector
- RandomPlayer
- Tit For Tat (TFT)
- TFT2 (variant)
